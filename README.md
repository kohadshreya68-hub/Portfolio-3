# Interactive Portfolio (Week 3 – JavaScript)

## Overview
A simple portfolio website demonstrating JavaScript features:
- DOM manipulation
- Events
- Form validation
- Dark/Light mode
- Image slider
- To-do list

## Project Goals
Make websites interactive using core JavaScript concepts.

## Setup
1. Download project
2. Keep files in structure:

index.html
style.css
script.js
images/

3. Open index.html in browser.

## Features Explained
- Dark mode saved in localStorage
- Form validation shows error/success messages
- To-do list (add/remove)
- Image slider switches images
- Reusable helper functions

## Testing
✔ Invalid email → shows error  
✔ Short message → shows error  
✔ Valid form → success  
✔ Dark mode stays after refresh  
✔ To-do list adds & deletes items  
✔ Slider moves forward/back
